# ATLAS question bank v1 (`content/questions/atlas-v1/`)

**Purpose:** ATLAS instrument items (stems, keys, routing metadata) for the adaptive engine when ATLAS is enabled. Versioned together (e.g. `bank.json` + locale packs) per study needs.

**Do not put here**

- PCMS F–V items (`global-behavioral-v2`, `cultural-adaptive-v1`, `universal`, etc.).
- Shared “merged” banks that mix PCMS and ATLAS items in one file without a documented migration (**ADR-002**).

**Governance:** **ADR-001** (paths), **ADR-002** (separate banks and scoring). [`docs/DECISIONS.md`](../../../docs/DECISIONS.md).
